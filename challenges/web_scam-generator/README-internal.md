### Scam generator

#### Requirements
- [docker](https://www.docker.com/)
- [docker-compose](https://docs.docker.com/compose/)
- UNIX environment

#### Run
`./run.sh`

#### Usage
Challenge will be available on:
- main page: `http://localhost:8093`
- report service: `http://localhost:4444`


#### Solver
`./solver.sh`
